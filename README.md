## About

FFMpeg Mobile AI Assistant is a powerful mobile application that combines the capabilities of FFmpeg with Google's Gemini AI to provide an intuitive interface for video and audio processing on mobile devices. This app allows users to perform complex media operations through a chat-based interface, making FFmpeg accessible to both beginners and advanced users.

## Features

- **AI-Powered Chat Interface**: Interact with FFmpeg through a conversational interface powered by Google's Gemini AI
- **File Management**: Import, process, and share media files directly from your device
- **Media Processing**: Execute FFmpeg commands for video/audio conversion, editing, and analysis
- **Real-time Feedback**: View logs and command outputs in real-time
- **Dark Mode UI**: Sleek, modern interface designed for media professionals
- **Cross-Platform**: Available for both iOS and Android devices

## Technology Stack

- **Flutter**: Cross-platform framework for mobile development
- **FFmpeg Kit**: Integration of FFmpeg libraries for media processing
- **Google Generative AI**: Utilizes Gemini AI for natural language processing
- **File Handling**: Implements file picking, sharing, and path management
- **Secure Storage**: Manages API keys and user preferences securely

## Getting Started

To run this project locally:

1. Ensure you have Flutter installed on your machine
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Configure your Gemini API key in the appropriate settings
5. Run the app using `flutter run`
